# Erster Prompt für Claude Code

Kopiere den folgenden Text und füge ihn in Claude Code ein:

---

Ich möchte eine persönliche Künstler-Website bauen. Lies bitte zuerst die CLAUDE.md — dort steht alles über mich, das Projekt und die Design-Richtung.

Bevor du mit dem Code anfängst, mach bitte Folgendes:

1. Schau dir die Bilder in `images/portfolio/`, `images/about/` und `images/shop/` an, um ein Gefühl für meine Kunst und die Farben zu bekommen
2. Schau dir die Referenz-Screenshots in `reference/` an — die Website soll eine Mischung aus diesen vier Stilen sein
3. Erstelle basierend darauf einen Design-Plan (Farben, Layout, Typografie) und zeig ihn mir kurz bevor du loslegst

Dann baue die Website als einzelne HTML-Datei (index.html) mit eingebettetem CSS und JS. Trenne erst in separate Dateien auf, wenn die Grundstruktur steht.

Nutze nach jeder größeren Änderung Puppeteer um einen Screenshot zu machen (Desktop 1440px und Mobile 375px) und prüfe visuell ob das Ergebnis hochwertig aussieht. Verbessere iterativ bis es professionell wirkt.

Wichtig:
- Neutrale, zurückhaltende Farben — die Kunst bringt die Farbe
- Kein generisches Template-Design
- Die Kunst muss im Vordergrund stehen
- Zweisprachig (DE/EN) mit Toggle
